CIDOC CRM — Perfil de Aplicação

Obra: Abaporu (2010) — Alexandre Mury

Versão: 1.0

Data: 17 de junho de 2026

Visão Geral

Este perfil estabelece a documentação ontológica completa da obra Abaporu (2010) segundo o padrão internacional CIDOC CRM (ISO 21127).

Seções Principais

1. Introdução

Definição do escopo, princípios de modelagem e objetivos do perfil.

2. Entidades Ontológicas

  • E22 Human-Made Object — A obra material
  • E12 Production — Evento de produção
  • E7 Activity — Atividade performática
  • E39 Actor — Agente produtor (Alexandre Mury)
  • E28 Conceptual Object — Conceitos autorais
  • E78 Curated Holding — Coleções museológicas

3. Arquitetura de Conhecimento

Grafo ontológico completo mostrando as relações entre entidades.

4. Mapeamento Semântico

Correspondência entre:

  • Schema.org (descoberta na Web)
  • CIDOC CRM (patrimônio cultural)
  • Wikidata (conhecimento estruturado)
  • JSON-LD (implementação)
  • RDF/Turtle (interoperabilidade)

5. Extensões Autorais (AMO)

Alexandre Mury Ontology — camada complementar para representar:

  • Conceitos propriedades
  • Regimes de Relação
  • Operações interpretativas específicas
