Desapropriação Imagética

URI: https://www.alexandremury.art/id/concept/desapropriacao-imagetica

Autoria intelectual: Alexandre Mury

Documentação primária: Da Blasfêmia à Canonização (2025); Desapropriação Teórica (2025)

A desapropriação imagética nomeia a operação pela qual uma imagem é deslocada de seu contexto simbólico de origem sem intenção de reproduzi-lo, citá-lo ironicamente ou negar sua validade histórica. O resultado não é paródia, homenagem, pastiche nem negação: é diferença instituída. A imagem se descola de sua matriz para adquirir autonomia simbólica própria.
